Descripcion de estructura y metodos

El servicio est√° dividido en 4 "sub servicios"


  • ClienteRest URL /cliente
  • CuentaRest URL /cliente/{id_cliente}/cuentas
  • MovimientosRest URL /cliente/{id_cliente}/cuentas/movimientos/{id_cuenta}
  • EstadoMovimientoRest URL /estadosMovimientos

Cada servicio consta de los siguientes metodos

  • ClienteRest @GET,@POST,@PUT,@DELETE
  • CuentaRest @GET,@POST,@PUT,@DELETE
  • MovimientosRest @GET, @POST
  • EstadoMovimientosRest @GET

Caracteristicas extra:

  • Se puede consultar el saldo de cada cuenta con GET a /cliente/{id_cliente}/cuentas/{id_cuenta}/saldo
  • Se puede obtener una lista con todos los movimientos en la base de datos con GET a /estadosMovimientos
  • Se puede obtener una lista de los movimientos discriminada por tipo con GET a /estadosMovimientos/{tipo}

Creacion de un cliente:

JSON = {"cuil":bigint,"nombre":string}


Creacion de una cuenta:

JSON = {"numero":bigint,"id":int}


Creacion de un movimiento:

JSON = {"estado":int,"id":int,"importe": long,"tipo":int}
